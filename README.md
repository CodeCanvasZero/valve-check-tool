# KDSHT阀体检测工具

## 功能说明
- 输入零件号自动检测是否需要打铅封孔和铭牌孔
- 本地使用次数记录（基于浏览器）
- 全局使用次数统计（基于免费API）
- 移动端友好设计

- v1.1.0 (2025-11-26): 增加使用次数记录功能
